API Description

ISO country code from coordinates

The first API will update when co-ordinates are set within a country. The default coordinates below are set to London.

Here are some more suggestions to try:

  • Berlin: (52, 13)
  • Cape Town: (-33, 18)
  • Delhi: (28, 77)

Weather from coordinates

This API will update when the coordinates are set within a monitored weather system such as a country. Here are some more suggestions for you to try but the above suggestions work fine here too.

  • Madrid: (40, 4)
  • New York: (41, -74)
  • Paris: (48, 2)

Hierarchies

Finally, the last API we are demonstrating will return places higher up in the hierarchy of a named place.

If a city is selected below the results will display the city itself as well as the country and continent of which it is a geographical part of.
